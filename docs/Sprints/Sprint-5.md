# Sprint 5: Refinamiento de Identidad Visual "Quiet Luxury"

## Objetivo
Ajustar la estética del sitio hacia un minimalismo sobrio y de categoría, alineado con el brief del cliente (OH! Buenos Aires vs Mundo OH!). Introducir la nueva paleta de colores y simplificar elementos visuales.

## Tareas Principales
- [ ] **Redefinición de Identidad en `globals.css`**:
    - Reemplazar la paleta basada en Rosa Viejo y Oro intenso.
    - Definir "Celeste OH!" (Sky Blue premium), Blanco y Negro como base.
    - Simplificar gradientes metálicos para un look más sobrio.
- [ ] **Actualización de Header & Footer**:
    - Refactorizar componentes para reflejar la nueva sobriedad.
    - Asegurar legibilidad y tracking tipográfico amplio (estilo Gotham).
- [ ] **Ajuste de MarketingSplash & ContactSection**:
    - Adaptar los formularios y pop-ups a la nueva estética.
    - Optimizar transiciones y animaciones para que se sientan "premium" pero discretas.

## Criterios de Aceptación
- El sitio ya no utiliza colores tierra/marrones.
- El color celeste se utiliza de manera elegante como acento.
- El diseño se siente más despejado y minimalista.
